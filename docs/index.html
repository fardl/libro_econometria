Sobre este proyecto

Este sitio acompaña el desarrollo progresivo del libro del curso. El documento se construye en LaTeX, se versiona con GitHub y se publica automáticamente para que los estudiantes siempre tengan acceso a la versión más reciente.

Contenido del libro

Capítulo 1

Introducción, motivación, preguntas empíricas y rol de los datos.

Capítulo 2

Modelo de regresión lineal simple y fundamentos de estimación.

Capítulo 3

Regresión múltiple, interpretación de coeficientes y supuestos.

Capítulo 4

Inferencia, pruebas de hipótesis y restricciones lineales.

Capítulo 5

Datos de corte transversal, limpieza de bases y aplicaciones dominicanas.

Capítulo 6

Introducción a inferencia causal y evaluación de políticas públicas.

Material para estudiantes
